termux的使用

Created2023-09-18|Updated2023-09-20|关于Linux
|Word count:2.2k|Reading time:8min|Post View:

介绍

termux 是一个安卓终端模拟器，具有 Linux 环境，具有最小基本系统，使用 APT 包管理器可以获得其它包。因此可以在 termux 实现几乎在 Linux 上的所有操作，比如可以使用 ssh 远程连接主机，使用 gcc g++ make 等编辑器，更或者是使用 MySQL 等等操作。

如何下载？

  • 在谷歌商店搜索 termux 并下载
  • 在浏览器中找一个靠谱的下载
  • 在 termux 的开源地址github上下载
  • 注：在 github 网络连接不稳定，可以使用科学上网来解决

基础

由于 termux 是基于 Linux 的最小基本系统，所以应该具有如下的 Linux 命令基础

  • Linux 基本命令

  • vim 基本操作（可以使用 nano 代替）

termux 基本操作

长按屏幕

显示菜单项（包括粘贴复制等选项）

左上角从左往右滑

显示隐藏式导航栏，可以新建，切换，重命名会话和调用弹入输入法

显示拓展功能按键

快捷键一览
  • Ctrl + A -> 将光标移动到行首
  • Ctrl + C -> 中止当前进程
  • Ctrl + D -> 注销终端会话
  • Ctrl + E -> 将光标移动到行尾
  • Ctrl + K -> 从光标删除到行尾
  • Ctrl + U -> 从光标删除到行首
  • Ctrl + L -> 清除终端
  • Ctrl + Z ->挂起（发送 SIGTSTP 到）当前进程
  • Ctrl + alt + C ->打开新会话（仅适用于黑客键盘）
  • 音量加 + E -> Esc 键
  • 音量加 + T -> Tab 键
  • 音量加 + 1 -> F1（音量增加 + 2 →F2…以此类推）
  • 音量加 + 0 -> F10
  • 音量加 + B -> Alt + B，使用 readline 时返回一个单词
  • 音量加 + F -> Alt + F，使用 readline 时转发一个单词
  • 音量加 + X -> Alt+X
  • 音量加 + W -> 向上箭头键
  • 音量加 + A -> 向左箭头键
  • 音量加 + S -> 向下箭头键
  • 音量加 + D -> 向右箭头键
  • 音量加 + L -> | （管道字符）
  • 音量加 + H -> 〜（波浪号字符）
  • 音量加 + U -> _ (下划线字符)
  • 音量加 + P -> 上一页
  • 音量加 + N -> 下一页
  • 音量加 + . -> Ctrl + \（SIGQUIT）
  • 音量加 + V -> 显示音量控制
  • 音量加 + Q -> 切换显示的功能键视
  • 音量加 + K -> 切换显示的功能键视图
自定义附加键
vi ~/.termux/termux.properties
#添加自定义键
extra-keys = [ \ ['ESC','|','/','HOME','UP','END','$','DEL'], \['TAB','CTRL','-','LEFT','DOWN','RIGHT','"','BKSP'] \ ]
#此操作是在输入法上端添加附加键
#在文档中所有快捷键都有注释
#HOME为首项，END为尾项

termux 目录环境结构

~ > echo $HOME
/data/data/com.termux/files/home

 ~ > echo $PREFIX
/data/data/com.termux/files/usr

 ~ > echo $TMPPREFIX
/data/data/com.termux/files/usr/tmp/zsh

#输入env可以查看各环境变量

基本软件安装

pkg install curl git vim wget

更改国内源

提高下载速度，可更改 termux清华大学源

手动修改
#下载vim文本编辑器
apt install vim
vim $PREFIX/etc/apt/sources.list
vim $PREFIX/etc/apt/sources.list.d/science.list
vim $PREFIX/etc/apt/sources.list.d/game.list
#或者
apt edit-source
#按i进入插入模式，将deb后的链接替换成http://mirrors.tuna.tsinghua.edu.cn/termux
自动配置
#图像，但可能会出错
pkg install termux-tools
#更新/安装termux-tools软件包
termux-change-repo
#使用GUI界面选择镜像源Mirrors by USTC/选择Mirrors by Tsinghua University

#自动替换
sed -i 's@^\(deb.*stable main\)$@#\1\ndeb
https://mirrors.tuna.tsinghua.edu.cn/termux/termux-packages-24 stable main@'
$PREFIX/etc/apt/sources.list
sed -i 's@^\(deb.*games stable\)$@#\1\ndeb
https://mirrors.tuna.tsinghua.edu.cn/termux/game-packages-24 games stable@'
$PREFIX/etc/apt/sources.list.d/game.list
sed -i 's@^\(deb.*science stable\)$@#\1\ndeb
https://mirrors.tuna.tsinghua.edu.cn/termux/science-packages-24 science  stable@'
$PREFIX/etc/apt/sources.list.d/science.list pkg update

至此，准备工作完成

美化

termux 内置 bash 这个 shell，但插件较少，可以通过安装 zsh 这个 shell 来进行美化。

#让termux能够访问内部存储，在Android6版本以上会弹出弹窗请求访问内部存储，同意即可
termux-setup-storage
#安装zsh，-y选项是默认yes不用再选择了
apt install zsh -y
#修改为默认shell
chsh -s /bin/zsh
#安装oh-my-zsh插件，这一步可能需要科学上网
sh -c "$(curl -fsSL https://github.com/Cabbagec/termux-ohmyzsh/raw/master/install.sh)"
#运行完这个脚本后会分别让你选择背景色和字体，可能看个人爱好，我推荐主题色14，字体默认（会在安装主题时更改）
#更改背景色/字体
chcolor #或~/.termux/colors.sh
chfont #或~/.termux/fonts.sh

#修改欢迎界面
vim $PREFIX/etc/motd
#将其中文字修改成自己想要的即可，如果想要字符画，可以安装figlet来生成
pkg install figlet
#使用方法：
figlet -f 字体 字符
#然后复制到motd即可
#若"figlet 字符"则会是默认字体

插件

zsh 之所以备受欢迎不仅仅是因为配色和主题，也有各种各样的插件

#autosuggestions插件
#根据自己的习惯自动提示接下来要使用的命令，按向右方向键自动补全
git clone https://github.com/zsh-users/zsh-autosuggestions $ZSH_CUSTOM/plugins/zsh-autosuggestions
#syntax-highlighting插件
#语法高亮
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git $ZSH_CUSTOM/plugins/zsh-syntax-highlighting
#在.zshrc文件中配置：
plugins=(其他插件 zsh-autosuggestions zsh-syntax-highlighting)
#重启生效

主题

推荐 powerlevel10k 主题
该主题会通过用户的选择来显示

git clone --depth=1 https://gitee.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
#在./zshrc文件中配置：
ZSH_THEME={powerlevel10k/powerlevel10k}
#安装引导界面推荐的字体

官方主题戳此获取，使用方法是将 ZSH_THEME 中修改成对应名称。

安装 Linux 发行版

termux 在手机没有 root 权限也可以使用，可以模拟出一个 root 环境
首先要安装 proot

官方做法

pkg install proot -y
#切换成模拟的root用户
termux-chroot

#安装套件
pkg install proot-distro
#查询帮助文档
proot-distro --help
#列出支持的Linux
proot-distro list
#安装，<alias>是Linux的别名，如我要安装Ubuntu20.04版本，则输入ubuntu-20.04
proot-distro install <alias>
#进入Linux环境
proot-distro login <alias>
#如果认为进入Linux的命令过长，可以设置别名，先进入.zshrc文件
alias 别名="proot-distro login <alias>"
#令上述命令生效
source ~.zshrc
#或者将该行命令写入sh脚本中
#创建sh脚本
touch 名称.sh
#然后粘贴进去，以后只要运行该脚本即可
